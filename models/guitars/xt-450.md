# 기타 모델 — XT-450

재사용 가능한 기타 모델 정의. rig에서 이 모델을 참조한다. → [[rigs]]

## 스펙

- **바디**: 확인 필요
- **넥**: 확인 필요
- **픽업 구성**: HSS
  - **넥**: 싱글코일 — 따뜻하고 둥근 클린, 블루스 리드
  - **미들**: 싱글코일 — 글래시한 펑크/커팅
  - **브릿지**: 험버커 — 두껍고 푸시 강한 록/하이게인
- **5-way 셀렉터** (표준 HSS 배선 가정 — 실제 배선이 다르면 이 파일 수정):
  1. 브릿지 험버커
  2. 브릿지 + 미들
  3. 미들
  4. 미들 + 넥
  5. 넥
- **코일 스플릿**: 확인 필요 (푸시-풀/스위치로 브릿지 험버커 스플릿 가능한지)
- **트레몰로/브릿지**: 확인 필요

## 픽업 선택 가이드 (패치 추천 시)

| 원하는 톤 | 추천 포지션 |
|-----------|-------------|
| 따뜻한 클린, 발라드 리드 | 넥 싱글 (pos 5) |
| 글래시 펑크, 커팅 리듬 | 미들 / 미들+넥 (pos 3~4) |
| 쿼킹(quack), 펑키 | 브릿지+미들 / 미들+넥 (pos 2,4) |
| 두꺼운 록 리듬, 하이게인 솔로 | 브릿지 험버커 (pos 1) |
| 록인데 살짝 트인 톤 | 브릿지 코일 스플릿 (스플릿 지원 시) |

  실기 확인 후 이 파일을 채운다.
