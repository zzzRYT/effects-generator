# GP-150 앰프 모델 레퍼런스

GP-150 팩토리 앰프 모델의 "GP-150 표기 이름 ↔ 모델링 원본 ↔ 캐릭터 ↔ 어울리는 곡/장르" 매핑.
이 파일이 톤 매칭의 핵심 데이터다.

> 상태: **앰프 모델 목록 입력 대기 중.**
> 기기/매뉴얼에 표시된 앰프 모델 이름을 받으면, 각 모델을 검색해 아래 형식으로 채운다.

## 형식

```markdown
## <GP-150 표기 이름>
- 모델링 원본: <실제 앰프 (예: Marshall JCM800)>
- 캐릭터: <게인대, EQ 성향, 질감>
- 장르/용도: <대표 장르>
- 추천 게인대: <0~10 중 범위>
- 대표 톤 예시: <이 톤으로 유명한 곡/밴드>
- GP-150 팁: <어울리는 픽업/캐비닛/주의점>
```

## 분류 가이드 (목록 채울 때 참고)

곡 톤 → 앰프 역추적이 빠르도록 캐릭터 버킷으로 묶어 관리:

| 버킷 | 대표 원본 | 쓰임 |
|------|-----------|------|
| 클린 (Fender 계열) | Twin Reverb, Deluxe | 펑크, 컨트리, 클린 아르페지오 |
| 빈티지 크런치 (Vox/AC) | AC30 | 브릿팝, 60s 록, 클래식 |
| 록 크런치 (Marshall) | Plexi, JCM800 | 하드록, 클래식록 리듬 |
| 모던 하이게인 | Mesa Rectifier, 5150 | 메탈, 모던록 |
| 부티크/엣지 | Friedman, Bogner | 모던 리드, 다용도 |
