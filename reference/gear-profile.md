# 장비 프로파일

내 셋업의 고정 입력. 모든 패치 추천은 이 장비를 기준으로 한다.

## 기타: Cort G250

- **바디**: 배스우드 — 밸런스 잡힌 중립적 톤, 미드 약간 부드러움
- **넥**: 메이플
- **픽업 구성**: HSS (Powersound)
  - **넥**: 싱글코일 — 따뜻하고 둥근 클린, 블루스 리드
  - **미들**: 싱글코일 — 글래시한 펑크/커팅
  - **브릿지**: 험버커 — 두껍고 푸시 강한 록/하이게인
- **5-way 셀렉터** (S-스타일 배선):
  1. 브릿지 험버커
  2. 브릿지 + 미들
  3. 미들
  4. 미들 + 넥
  5. 넥
- **푸시-풀 톤 노브**: 브릿지 험버커 **코일 스플릿** → 싱글코일 톤 확보
- **2-포인트 트레몰로**: return-to-zero, 다이브/벤딩

### 픽업 선택 가이드 (패치 추천 시)

| 원하는 톤 | 추천 포지션 |
|-----------|-------------|
| 따뜻한 클린, 발라드 리드 | 넥 싱글 (pos 5) |
| 글래시 펑크, 커팅 리듬 | 미들 / 미들+넥 (pos 3~4) |
| 쿼킹(quack), 펑키 | 브릿지+미들 / 미들+넥 (pos 2,4) |
| 두꺼운 록 리듬, 하이게인 솔로 | 브릿지 험버커 (pos 1) |
| 록인데 살짝 트인 톤 | 브릿지 코일 스플릿 |

## 멀티이펙터: Valeton GP-150

- **모델링**: 2세대 HD 모델링, 200+ 이펙트
- **모듈**: 최대 12개 동시, 순서 자유 배치
- **캐비닛**: IR 로딩(서드파티 20개) + NAM(SnapTone, 팩토리 50 포함 최대 100)
- **컨트롤**: 풋스위치 2 + 익스프레션 페달 (→ 곡 중 사운드 전환에 활용)
- **기타**: 루퍼(180초), 드럼머신, 48kHz/24bit, USB 오디오 인터페이스

### 기본 시그널 체인 규칙

표준 배치 순서(특별한 이유 없으면 이대로):

```
기타 → [게이트] → [드라이브/부스트] → [앰프] → [캐비닛/IR]
     → [모듈레이션] → [딜레이] → [리버브] → 출력
```

- **게이트**: 하이게인일 때만 추가
- **드라이브/부스트**: 앰프 앞 (게인 푸시) 또는 솔로 부스트용
- **공간계(딜레이/리버브)**: 항상 체인 뒤쪽
- **모듈레이션**(코러스/플랜저)는 보통 앰프 뒤, 딜레이 앞
