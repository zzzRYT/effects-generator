# GP-150 이펙트 블록 레퍼런스

곡별 패치를 구성할 때 앰프 외에 얹는 이펙트 블록들의 캐릭터와 용도 정리.
GP-150은 200+ 이펙트가 있으므로, 여기엔 곡 톤 매칭에 자주 쓰는 핵심 카테고리만 정리한다.
정확한 블록 이름은 기기 화면 기준으로 채워 넣으며, 모르는 이름은 검색으로 보강한다.

> 채우는 법: 카테고리별로 GP-150에 들어있는 대표 모델을 적고, 어떤 곡/장르에 쓰는지 메모.

## 다이내믹 / 게이트

- **노이즈 게이트**: 하이게인 시 험 차단. 임계값(threshold) 너무 높이면 음 끊김.
- **컴프레서**: 클린 펑크 커팅, 컨트리 치킨피킹, 어택 다듬기. 스튜디오 클린에 살짝.

## 드라이브 / 부스트 (앰프 앞)

- **오버드라이브 (TS류)**: 미드 푸시, 앰프 게인 더 밀기. 솔로 부스트의 기본.
- **디스토션 (RAT/Big Muff류)**: 독립적 거친 게인. 펑크/그런지/스토너.
- **클린 부스트**: 게인 변화 없이 레벨만 올림 → 솔로 도드라지게.

## 모듈레이션 (앰프 뒤, 딜레이 앞)

- **코러스**: 80년대 클린, 더블링 두께. 브릿팝/뉴웨이브.
- **플랜저 / 페이저**: 펑크 리듬, 사이키델릭, 슈게이즈 텍스처.
- **트레몰로**: 서프, 빈티지 흔들림.

## 딜레이 (체인 뒤쪽)

- **아날로그 딜레이**: 따뜻하고 어두운 반복, 짧은 슬랩백.
- **디지털 딜레이**: 또렷한 반복, U2식 점음표(dotted-eighth) 리드.
- **테이프 딜레이**: 빈티지 흔들림 + 따뜻함.
- 곡 BPM에 맞춰 탭템포 / 노트값(1/4, 점8분) 지정.

## 리버브 (가장 뒤)

- **스프링**: 서프, 빈티지 클린.
- **룸/플레이트**: 자연스러운 공간감, 범용.
- **홀/슈머**: 앰비언트, 슈게이즈, 포스트록 — 깊고 길게.

## 기타

- **와우**: 익스프레션 페달에 할당 → 펑크/솔로.
- **볼륨 페달**: 익스프레션 페달 → 볼륨 스웰.
- **옥타버 / 피치**: 두께 보강, 신스 느낌.

## 사용 원칙

- 곡에 실제로 들리는 이펙트만 넣는다. 안 들리면 빼는 게 톤에 이롭다.
- 공간계(딜레이/리버브) 믹스는 과하지 않게 — 합주에서 묻힌다.
- 곡 중 사운드가 바뀌면 패치의 "스위칭 플랜" 섹션에 풋스위치 할당 명시.
