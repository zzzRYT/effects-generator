# GP-150 톤 빌더 설계

날짜: 2026-06-18

## 목표

일렉기타 연주 시 곡마다 적절한 앰프/이펙트 사운드를 GP-150에서 빠르게 세팅하기 위한
2층 구조 시스템. 곡 이름을 주면 그 곡의 실제 톤을 조사해 GP-150 패치로 매핑하고,
곡별 패치를 라이브러리로 누적한다.

## 장비 (고정 입력)

- 기타: Cort G250 (HSS 구성, Powersound 픽업)
- 멀티이펙터: Valeton GP-150
  - 200+ HD 모델링 이펙트, 최대 12개 모듈 자유 배치
  - IR 캐비닛 로딩(서드파티 20개) + NAM(SnapTone, 팩토리 50 포함 최대 100)
  - 풋스위치 2 + 익스프레션 페달, 루퍼, 드럼머신, USB 오디오 인터페이스

## 핵심 결정

| 항목 | 결정 |
|------|------|
| 산출물 형태 | 레퍼런스 데이터(층1) + tone-builder 스킬(층2) |
| 입력 방식 | 곡 이름/아티스트 → 실제 톤 조사 후 GP-150 매핑 |
| 추천 범위 | 풀 시그널 체인 (앰프 + 캐비닛 + 핵심 이펙트) |
| 저장 방식 | 곡별 패치 파일 1개 + INDEX.md 누적 |
| 앰프 목록 출처 | 사용자가 기기/매뉴얼 목록 제공, 캐릭터는 검색으로 보강 |
| 사운드 전환 | 풋스위치 기반 스위칭 플랜 (선택 섹션) |

## 레퍼런스 데이터 형식 (gp150-amps.md)

각 팩토리 앰프 모델을 다음 구조로 정리:

```markdown
## Brit 800   (GP-150 표기 이름)
- 모델링 원본: Marshall JCM800
- 캐릭터: 미드 게인, 거친 크런치, 미드레인지 강한 록 사운드
- 장르/용도: 하드록, 펑크, 클래식록 리듬
- 추천 게인대: 5~8
- 대표 톤 예시: AC/DC, Guns N' Roses
- GP-150 팁: 험버커 궁합, 캐비닛 4x12 추천
```

핵심: "GP-150 이름 ↔ 실제 앰프 ↔ 캐릭터 ↔ 어울리는 곡/장르" 매핑.

## tone-builder 스킬 동작

1. 곡 톤 조사 — 원곡의 앰프/이펙트, 장르, 게인 정도, 특징 이펙트 파악
2. 앰프 매칭 — 톤 캐릭터를 gp150-amps.md와 대조, 가까운 모델 1~2개 선택
3. 풀 체인 구성 — 앰프+캐비닛+이펙트, 구체적 노브 값, G250 픽업 포지션
4. 곡별 패치 파일 저장 — patches/<artist>-<song>.md
5. INDEX 갱신

불확실성 처리: 톤을 못 찾거나 확신 낮으면 추측으로 채우지 않고 "확신 낮음" 표시 + 대안 제시.

## 패치 파일 항목

1. 시그널 체인 (블록 순서)
2. 모듈별 노브 값
3. G250 픽업 포지션
4. 스위칭 플랜 (선택) — 풋스위치 2개로 벌스/솔로/브릿지 사운드 전환
5. 왜 이 앰프인지 + 연주 팁

## 구축 순서

1. gear-profile.md (G250 스펙 검색 포함)
2. gp150-amps.md — 사용자가 앰프 모델 목록 제공 → 각 모델 검색 보강
3. gp150-effects.md
4. tone-builder/SKILL.md
5. 검증: 톤이 잘 알려진 곡 2~3개로 테스트 → 실제 GP-150 입력 → 귀로 듣고 피드백 → 보정
